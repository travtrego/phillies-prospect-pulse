# Phillies Prospect Pulse

A continuously updated database and dashboard for Philadelphia Phillies prospects.

## Phase 1: Player master database

The first phase intentionally focuses only on durable player information:

- MLB player ID
- Full name
- Primary position
- Birth date and age
- Bats / throws
- Height / weight
- Country of birth
- Current level
- MLB Pipeline organizational rank, when applicable
- Estimated MLB arrival year, when published
- Active/inactive status within the Phillies organization
- Source and last-verified timestamps

Stats, injuries, news, scouting reports, and affiliate-history automation will be added after the player master database is stable.

## Initial source priority

1. MLB / MiLB official prospect and player records
2. Official Phillies organization and affiliate rosters
3. Other public lists only as cross-checks

The MLB player ID is the canonical identifier whenever available, preventing duplicate records when names or assignments change.
